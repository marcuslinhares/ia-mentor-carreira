## 📋 Sobre o Projeto

Sistema de detecção de fraudes em transações financeiras usando Machine Learning.

- ✅ **Dataset real:** Credit Card Fraud Detection (284.807 transações)
- ✅ **Análise exploratória** com visualizações
- ✅ **Pré-processamento:** padronização + SMOTE para balanceamento
- ✅ **Modelos:** Regressão Logística e Random Forest
- ✅ **Avaliação:** matriz de confusão, curva ROC, métricas por classe
